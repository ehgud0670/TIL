# chapter 10
## 1차원 배열

### 10-1 배열의 이해 & 배열의 선언 & 초기화
1. 배열이란?
: 가장 많이 쓰이는 자료구조 중 하나 
```c
int arr[100]; // 길이가 100인 int형 1차원 배열: 정수 100개 저장 가능.
// sizeof(arr) = 100*4 =400 byte
```
2. 배열의 선언의 필요 요소
   1. 배열을 이루는 요소의 자료형 (int)
   2. 배열의 이름 (arr)
   3. 배열의 길이 ([100])
